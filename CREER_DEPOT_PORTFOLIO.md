# Créer un nouveau dépôt pour votre portfolio

## Pourquoi créer un nouveau dépôt ?

Votre dépôt actuel `My-data-science-project` contient vos projets de data science.
Pour le portfolio, il est préférable d'avoir un dépôt séparé avec :
- Un nom clair : `portfolio` ou `celestin-portfolio`
- Un dépôt dédié au site web
- Une URL plus simple pour votre site

## Étape à suivre :

1. Allez sur : **https://github.com/new**
2. **Repository name** : Tapez `portfolio`
3. **Description** : "Portfolio personnel - Celestin Signe Sighano"
4. Cochez **Public**
5. **NE COCHEZ RIEN D'AUTRE**
6. Cliquez sur **"Create repository"**

Après la création, vous aurez une URL comme :
```
https://github.com/SighanoCel/portfolio.git
```

**Revenez avec cette nouvelle URL et je vous aiderai à envoyer les fichiers !**

---

## Si vous préférez utiliser le dépôt existant

Si vous voulez vraiment utiliser `My-data-science-project`, c'est possible aussi.
Dans ce cas, je vous aiderai à mettre les fichiers du portfolio dans un sous-dossier ou à la racine.
