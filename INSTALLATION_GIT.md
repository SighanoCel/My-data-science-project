# Installation de Git sur Windows

## Méthode 1 : Téléchargement direct (Recommandé)

1. Allez sur le site officiel : https://git-scm.com/download/win
2. Le téléchargement devrait commencer automatiquement
3. Une fois téléchargé, double-cliquez sur le fichier `.exe`
4. Suivez les étapes de l'installation :
   - Cliquez sur "Next" plusieurs fois (les paramètres par défaut sont bons)
   - À l'étape "Choosing the default editor", vous pouvez garder "Nano editor" ou choisir "Visual Studio Code" si vous l'avez installé
   - Continuez avec "Next" jusqu'à "Install"
   - Cliquez sur "Install" et attendez la fin de l'installation
   - Cliquez sur "Finish"

5. Vérifiez l'installation :
   - Ouvrez PowerShell (ou un nouveau terminal)
   - Tapez : `git --version`
   - Vous devriez voir quelque chose comme : `git version 2.xx.x`

## Méthode 2 : Via GitHub Desktop (Plus simple)

Si vous préférez une interface graphique :

1. Allez sur : https://desktop.github.com/
2. Téléchargez GitHub Desktop
3. Installez-le (il installe aussi Git automatiquement)
4. Connectez-vous avec votre compte GitHub

## Après l'installation

Redémarrez PowerShell/Terminal pour que Git soit reconnu.
