# ✅ Prochaines étapes pour mettre votre site en ligne

## Ce qui est déjà fait ✅
- ✅ Git est installé et configuré
- ✅ Tous vos fichiers sont sauvegardés dans Git (commit créé)
- ✅ 25 fichiers sont prêts à être envoyés

## Étape suivante : Créer le dépôt sur GitHub

### 1. Créer le dépôt GitHub
1. Allez sur : **https://github.com/new**
2. **Repository name** : Tapez `portfolio` (ou un autre nom simple)
3. **Description** : (optionnel) "Portfolio personnel"
4. Choisissez **Public** (important pour l'hébergement gratuit)
5. **NE COCHEZ RIEN** (pas de README, pas de .gitignore, pas de license)
6. Cliquez sur **"Create repository"** (bouton vert)

### 2. Copier l'URL du dépôt
Après avoir créé le dépôt, GitHub vous montrera une page. 
**COPIEZ l'URL** qui ressemble à :
```
https://github.com/VOTRE-USERNAME/portfolio.git
```

### 3. Revenir ici avec l'URL
Une fois que vous avez l'URL, dites-moi simplement :
- Votre nom d'utilisateur GitHub
- Le nom du dépôt que vous avez créé

Ou donnez-moi l'URL complète, et je vous aiderai à connecter et envoyer vos fichiers !

---

## Commandes que nous exécuterons ensuite

Une fois que vous avez créé le dépôt GitHub, nous ferons :

```powershell
git remote add origin https://github.com/VOTRE-USERNAME/NOM-DU-DEPOT.git
git branch -M main
git push -u origin main
```

Et ensuite nous activerons GitHub Pages pour mettre votre site en ligne !
